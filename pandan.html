Can you make the history be further back than 120 days?

The history is intentionally limited. Pandan is about time awareness right now, not analyzing past behavior. If you need more stats and graphs, check out the system Screen Time feature.

Can you make the app available on iOS too?

This is unfortunately not possible as iOS does not let an app continuously check whether the user is idle.

How can I play a custom sound as a notification?

Make a shortcut that uses the Play Sound action and then select that shortcut in the Pandan settings.

If you have my Dato app installed (even the free trial), you could use one of the sounds it provides. Find the Get Sound action. Example

How can I show an alert as a notification?

How can I run multiple shortcuts as a notification?

Pandan lets you run a single shortcut as a notification, but a shortcut can run other shortcuts. So make the shortcuts you want to run, make another shortcut that uses the ‚ÄúRun Shortcut‚Äù action to run the other shortcuts, and then select this shortcut in Pandan.

Can you support iCloud sync?

No. iCloud sync is unreliable and creates too many support issues, especially for a free app.

How can I use this as an eye-break reminder?

Set Pandan to remind you to take a break. Make a shortcut in the Shortcuts app that uses the ‚ÄúWait‚Äù action and set it to the wanted duration of the break. After this action, you can either add a ‚ÄúPlay Sound‚Äù or ‚ÄúShow Alert‚Äù action to notify you when the break is done. Then select this shortcut in the Pandan settings.

How can I disable Pandan while using certain apps?

You can use the Shortcuts app for this. Make a shortcut that uses the ‚ÄúSet Enabled State‚Äù action that Pandan provides. To automate running this shortcut when certain apps are active, you will need the Shortery app. In the Shortery app, choose the ‚ÄúApplication‚Äù trigger type, and then the ‚ÄúActive‚Äù or ‚ÄúInactive‚Äù option.

Note that Pandan already disables itself when you are inactive.

How can I show the total active duration today as a countdown?

Get this shortcut (requires the Actions and One Thing apps) and select it in the Pandan settings. The countdown will update every time Pandan shows a notification. If you want the countdown to update more often, you will need to download the Shortery app and set up an interval trigger.

How can I start a new session from the command-line?

Make a shortcut that uses the New Session action and run that shortcut from the command-line.

Tip: Shortcuts can be a bit slow sometimes. You can use this command to have it not wait for the shortcut to finish running: (&>/dev/null shortcuts run ShortcutName &)
