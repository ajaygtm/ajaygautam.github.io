1.4.4

June 6, 2023
  • There is a bug in ChatGPT when the window is narrow (so narrow that the sidebar hides), the text field changes behavior to add newlines when pressing Return instead of submitting the message. QuickGPT used to work around this bug, however, this workaround now causes a weird problem where messages end up empty after submitting. I tried to fix the workaround, but it only fixed it for some users. This workaround is now removed. If you find the ChatGPT bug annoying, I recommend sending a bug report to the ChatGPT team: support@openai.com
